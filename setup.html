ASRP Setup Wizard

Configure your AI-powered research platform in minutes. All configuration is generated locally — no data leaves your browser.

1

API Keys

Provide at least 2 API keys for redundancy. The system will automatically assign optimal models to each agent role. An OpenRouter key is recommended as fallback.

💡 Best Practice: Multiple providers ensure uninterrupted research — if one service goes down, agents automatically switch to backup providers.
Opus for deep reasoning, Sonnet for code
Flash for fast searches and routine tasks
Available as alternative for any role
Automatic fallback if primary provider fails
2

Agent Configuration

Name your agents and configure their communication channel. Each agent has a specialized role in the research workflow.

Theorist
Hypotheses, reasoning, paper writing
Engineer
Code, computation, data pipelines
Reviewer
Independent review (read-only access)
Librarian
Literature search, references
ITDoctor
Monitoring, backups, self-healing
